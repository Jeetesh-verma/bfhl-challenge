## ⚡ TL;DR - Get Started Now

### Step 1: Update Your Info
Open `backend/processor.js` and change (around line 285):
```javascript
user_id: "yourname_ddmmyyyy",        // e.g., "johndoe_17091999"
email_id: "your.email@college.edu",  // Your actual college email
college_roll_number: "XXCSXXXX"      // Your college roll number
```

### Step 2: Start Backend
```bash
cd backend
npm install
npm start
```
Wait for message: "✓ Server running on port 5000"

### Step 3: Start Frontend (New Terminal)
```bash
cd frontend
npx http-server -p 3000
```
Or: `python -m http.server 3000`

### Step 4: Test in Browser
Open: http://localhost:3000
Click "Load Example" → Click "Process Hierarchies" → ✅ See results!
